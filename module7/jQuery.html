Pengertian jQuery

jQuery adalah library JavaScript yang dirancang untuk mempermudah penulisan kode JavaScript, terutama dalam manipulasi HTML, event handling, animasi, dan AJAX. Dengan jQuery, penulisan kode menjadi lebih singkat dan mudah dipahami.
